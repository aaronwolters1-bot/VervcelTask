# ListFlow

ListFlow is a static, mobile-first list app that deploys directly to Vercel with no build step.

## Local files

- `index.html`: list app structure and controls
- `styles.css`: responsive styling for the mobile list UI
- `app.js`: add, complete, reorder, and save list items in the browser
- `manifest.webmanifest`: install metadata
- `sw.js`: offline cache
- `vercel.json`: Vercel configuration

## Deploy to Vercel

1. Create a GitHub repository with these files.
2. Import the repository into Vercel.
3. Accept the defaults. No framework preset is required.
4. Open the deployed URL on your iPhone in Safari.
5. Use Share > Add to Home Screen.

## Features

- Create and switch between named lists
- Add new list items quickly
- Mark items complete without hiding them
- Reorder items with up and down controls
- Add optional due dates to items
- Delete items you no longer need
- Save data locally in the browser
